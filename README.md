# 📖 Project Overview

Financial institutions generate thousands of transactions every day. Raw data alone provides limited value unless it is transformed into meaningful insights that support decision-making.

This project demonstrates how **Microsoft Excel** can be used as a **Business Intelligence (BI)** tool to clean, analyze, summarize, and visualize financial data through an interactive executive dashboard.

The dashboard enables users to monitor key financial metrics, analyze loan performance, evaluate portfolio risk, and explore business trends using dynamic filters.

---

# 🎯 Business Problem

Large financial datasets are difficult to interpret in spreadsheet format.

Business managers require a centralized dashboard capable of answering questions such as:

- Which bank has the highest transaction volume?
- What percentage of loans are approved?
- Which loans belong to the high-risk category?
- What is the average interest rate?
- Which loan status has the largest customer base?
- What percentage of loans have matured?

This dashboard provides those answers instantly.

# 📈 Key Business Insights

The dashboard highlights several important findings:

- High-risk loans account for approximately **64%** of total exposure.
- Pending loans exceed approved loans.
- **CHASE** records the highest transaction amount.
- Expired loans represent approximately **79%** of the portfolio.
- High-risk exposure significantly outweighs low-risk exposure.
